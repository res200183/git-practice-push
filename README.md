\# Git Remote Workflow Demo



This repository demonstrates basic Git operations when working with a remote repository on GitHub.



\## Topics covered



\- Initializing a Git repository

\- Creating commits

\- Connecting a local repository to GitHub using `origin`

\- First `git push`

\- Setting an upstream branch

\- Handling rejected pushes

\- Using `git pull` to synchronize changes

\- Resolving merge conflicts

\- Pushing resolved changes back to GitHub



\## Example commands used



git init

git add .

git commit -m "first commit"

git remote add origin <repo-url>

git push -u origin master

git push

git pull

git add hello.txt

git commit -m "resolve merge conflict"

git push



\## Purpose



This repository was created as a practice project to demonstrate understanding of Git remote workflows and conflict resolution.
